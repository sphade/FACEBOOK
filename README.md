This is my version of facebook (a facebook clone)
Created with Next.js (server side rendering)
## functionalities
auth users are able to perform the following action
- post a picture
- post a caption
- comment on a photo

## stack

```bash
NEXT.JS
NEXTAUTH
REDUX
TAILWIND.CSS
FACEBOOK API
FIREBASE STORAGE
FIRBASE FIRESTORE
```
